Components can declare collisions, which are objects that will act as hitboxes. You can mix and match different collision types (points, boxes,cubes, polygons...) inside the collision object, each one will be represented as an array of objects, like this: 

    collision: {
            point: [
                { "#tag": "characterFeet", x: 0, y: 0},
            ],
            rectangle: [
                { "#tag": "bodyCollision", x: -50, y: -100, w: 100, h: 100 },
            ]
    }

As you can see, each collision object has the following fields:

  • #tag: this is an unique indentifier that will allow you to use to modify your collider from your code.
  • x: The x position of the collision object, relative to the object position.
  • y: The y position of the collision object, relative to the object position.
  • There can be more fields, depending on the type of the coliision object.
