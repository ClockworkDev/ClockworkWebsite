Project structure

In order to develop a Clockwork game, first you need to create a project. A project is a directory that will include all your files and must have a file named manifest.json where some basic information is stored. Clockwork projects can include different types of files:
  • Manifest
  • Components
  • Spritesheets
  • Levels
  • Other files (images, audio...)

You can create an starter project with the Create Clockwork project command in VS Code extension or running clockwork init [projectName] with the CLI tools.

If you are looking for a step-by-step guide for making your own game from scratch, take a look at the tutorials.
