Packages

You can use the Clockwork Package Manager to use existing packages to implement functionality in your game. If you have the CLI tools installed, you can run clockwork list to see the list of published packages, and run clockwork add 'package name' to add that package as one of the dependencies to your game.

One you have added a package, you can run the Browse Clockwork package documentation command in the Visual Studio Code extension to see the documentation for that package. Packages will usually declare one or several components that you need to add to your level to use.

You can also use the CLI tools to publish your own packages so they can be used by other developers!
