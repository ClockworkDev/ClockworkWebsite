The engine object exposes more methods, but you should not need them unless you have very, very specific needs (that means that if you can find another way to it, it will be probably be more elegant and safe) since they take care of the functionality that is already handled by the Clockwork Runtime. Here they are:

  • pause: This function pauses the whole engine, so #loop events will stop happening.
  • getRenderingLibrary: This function returns the rendering library used by the game, so you can manually call the methods it exposes if needed.
