There is a Visual Studio Code extension that builds on top of the CLI tools and incorporates additional functionality, like debugging capabilities.

It is available in the VS Code marketplace and you can install it using the ext install clockwork command inside the editor.

It provides commands for the following actions:

  • Creating an starter Clockwork project (F1 > Create Clockwork Project)
  • Packaging the project into a .cw file (F1 > Build Clockwork Project)
  • Packaging a project and deploying it to the runtime (F1 > Deploy Clockwork Project)
  • Debugging the game (Deploying it and then pressing F5)

The debugger allows to set breakpoints that will be triggered once the selected event handler is executed, and it shows the object and engine variables, the event stack and allows to evaluate JavaScript expressions.
