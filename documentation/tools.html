Tools

There are two ways to develop Clockwork games:
  • CLI tools: This is a multiplatform command line program that allows you to develop Clockwork games while using your favorite code editor or IDE.
  • VS Code Extension: There is a Visual Studio Code extension that builds on top of the CLI tools and incorporates additional functionality, like debugging capabilities. This is the recommended way to build Clockwork games

Actually, that is not entirely true! There is a third way to make games, and that is using your own tools. You can take a look at the CLI tools NPM package and use the functions it exposes to create your own tools or IDE plugins.
